# 🌱 GrowEasy Smart Import

An AI-powered CRM Lead Import System that intelligently imports CSV files, validates data, maps fields automatically, and stores leads into the CRM with minimal manual effort.

---

## 🚀 Features

- 📂 Drag & Drop CSV Upload
- 🤖 AI-Powered Field Mapping
- 📊 Dataset Analysis
- ✅ Data Validation
- 👀 Preview Imported Records
- 📈 Import Summary Dashboard
- 🗂 CRM Lead Management
- ⚡ Fast Import Process
- 🔔 Toast Notifications
- 🎨 Modern Responsive UI

---

## 🛠 Tech Stack

### Frontend

- Next.js 16
- React
- TypeScript
- Tailwind CSS
- Lucide React
- React Dropzone
- Sonner

### Backend

- Node.js
- Express.js

- CSV Parser

## ⚙ Installation

### Clone Repository

```bash
git clone https://github.com/yourusername/groweasy-smart-import.git

cd groweasy-smart-import
```

---

## Frontend Setup

```bash
cd frontend

npm install

npm run dev
```

Frontend runs on

```
http://localhost:3000
```

---

## Backend Setup

```bash
cd backend

npm install

npm start
```

Backend runs on

```
http://localhost:5000
```
